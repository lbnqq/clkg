## 📊 技术实现

### 文件结构

```
第23轮新增文件 (2个):
├── zootopia-seasonal.js     (~8KB) - 季节性主题系统
└── zootopia-seasonal.css    (~4KB) - 轻量级样式
```

### API接口

```javascript
// 获取当前季节
ztGetSeason()  // 返回: 'spring' | 'summer' | 'autumn' | 'winter'

// 手动切换季节
ztSetSeason('spring')

// 高级API
ZtSeasonal.getCurrentSeason()      // 获取当前季节
ZtSeasonal.getSeasons()            // 获取所有季节配置
ZtSeasonal.setSeason('summer')     // 切换到夏季
ZtSeasonal.getCurrentEvent()       // 获取当前节日
ZtSeasonal.showEventBanner({...})  // 显示节日横幅
ZtSeasonal.refresh()               // 刷新季节系统
```

### CSS变量

```css
:root {
    --zt-season-primary: #FF9F43;  /* 季节主色 */
    --zt-season-accent: #FF6B6B;   /* 季节强调色 */
}
```

## 🎨 使用场景

### 自动模式

系统自动检测当前季节和节日，应用对应主题：

```javascript
// 自动初始化，无需任何代码
// 访问网站时自动生效
```

### 手动模式

用户可以手动切换季节：

```javascript
// 切换到春季
ztSetSeason('spring')

// 切换到冬季
ztSetSeason('winter')
```

### 开发者模式

显示自定义节日横幅：

```javascript
ZtSeasonal.showEventBanner({
    name: '我的生日',
    icon: '🎂',
    theme: 'spring',
    isMajor: true
});
```

## 📝 配置说明

### 启用/禁用

**默认启用**，无需额外配置。

### 自定义季节消息

编辑 `zootopia-seasonal.js` 中的 `SEASONS` 配置：

```javascript
spring: {
    messages: [
        '自定义春季消息1',
        '自定义春季消息2'
    ]
}
```

### 添加自定义节日

编辑 `SPECIAL_EVENTS` 配置：

```javascript
'MM-DD': {
    name: '我的节日',
    icon: '🎉',
    theme: 'spring',
    isMajor: false
}
```
