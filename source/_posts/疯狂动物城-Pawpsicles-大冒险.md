### 🍦 什么是 Pawpsicles？

Pawpsicles 是尼克的创新产品：

| 特点 | 描述 |
|------|------|
| **原料** | 巨型冰棍 + 老鼠冰棍 |
| **成本** | 几乎为零（"捡"来的） |
| **售价** | $2 - $15 |
| **目标客户** | 体型各异的动物 |

### 💰 商业模式解析

```javascript
// Pawpsicles 商业模式
const pawpsicleBusiness = {
  supplier: "Mr. Big's organization",
  cost: 0,
  salePrice: 15,
  dailySales: 200,
  netProfit: function() {
    return this.dailySales * this.salePrice - this.cost;
  }
};

console.log("每日净收入: $" + pawpsicleBusiness.netProfit());
```
