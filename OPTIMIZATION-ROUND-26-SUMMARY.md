## 🚀 使用指南

### 基础使用

**步骤1**: 访问网站，点击右下角🃏按钮

**步骤2**: 查看收藏簿和收集进度

**步骤3**: 点击任意卡牌查看详情

**步骤4**: 完成解锁条件获得新卡牌

**步骤5**: 收集全部12张卡牌！

### 高级用法

```javascript
// 显示收藏簿
ztShowCardCollection()

// 检查卡牌
if (ztHasCard('judy')) {
    console.log('已拥有朱迪卡牌')
}

// 获取卡牌信息
const card = ztGetCard('judy')
console.log(card.name)  // 朱迪·霍普斯
console.log(card.stats) // { speed: 95, courage: 100, ... }

// 获取收集进度
const progress = ztGetCollectionProgress()
console.log(`${progress.unlocked}/${progress.total} (${progress.percentage}%)`)
```

### API调用

```javascript
// 高级API
ZTCards.showCollection()  // 显示收藏簿
ZTCards.has('judy')       // 检查卡牌
ZTCards.get('judy')       // 获取卡牌
ZTCards.getAll()         // 获取所有卡牌
ZTCards.getProgress()    // 获取进度
ZTCards.getStats()       // 获取统计
ZTCards.add('judy')       // 添加卡牌
ZTCards.showDetail('judy')// 显示详情
```
