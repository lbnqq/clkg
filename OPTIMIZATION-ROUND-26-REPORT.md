```javascript

    legendary: {

```

### API接口

```javascript
// 显示收藏簿
ztShowCardCollection()

// 检查卡牌
ztHasCard('judy')

// 获取卡牌信息
ztGetCard('judy')

// 获取收集进度
ztGetCollectionProgress()

// 高级API
ZTCards.showCollection()  // 显示收藏簿
ZTCards.has('judy')       // 检查卡牌
ZTCards.get('judy')       // 获取卡牌
ZTCards.getAll()         // 获取所有卡牌
ZTCards.getProgress()    // 获取进度
ZTCards.getStats()       // 获取统计
ZTCards.add('judy')       // 添加卡牌
ZTCards.showDetail('judy')// 显示详情
```

## 🎨 使用场景

### 1. 查看收藏

**场景**: 访客可以查看自己收集的卡牌

**操作**:
1. 点击右下角🃏浮动按钮
2. 查看收集进度和统计
3. 点击任意卡牌查看详情
4. 阅读角色台词和描述

### 2. 收集卡牌

**场景**: 通过互动解锁新卡牌

**解锁条件**:
- 访问网站3天 → 朱迪
- 评论5次 → 尼克
- 阅读10篇文章 → 闪电
- 签到3次 → 克拉豪斯
- 分享3篇文章 → 芬尼克

### 3. API调用

```javascript
// 检查是否拥有卡牌
if (ZTCards.has('judy')) {
    console.log('已拥有朱迪！')
}

// 获取收集进度
const progress = ZTCards.getProgress()
console.log(`已收集 ${progress.unlocked}/${progress.total}`)

// 获取卡牌详情
const card = ZTCards.get('judy')
console.log(`${card.name}: ${card.quote}`)
```
