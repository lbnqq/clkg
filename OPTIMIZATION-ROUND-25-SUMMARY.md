## 🚀 使用指南

### 基础使用

**步骤1**: 访问网站，点击右下角👮按钮

**步骤2**: 填写罚单信息（可留空随机）

**步骤3**: 选择违规类型和地点

**步骤4**: 点击"生成罚单"

**步骤5**: 查看罚单预览，选择打印/分享/保存

### 高级用法

```javascript
// 快速生成罚单
ztGenerateZPDTicket({
    violator: '朱迪',
    type: 'fun',
    location: 'ZPD总部',
    notes: '忘记吃甜甜圈了'
})
```

### API调用

```javascript
// 显示生成器
ZPDTicket.showGenerator()

// 生成罚单
ZPDTicket.generate()

// 打印罚单
ZPDTicket.print()

// 分享罚单
ZPDTicket.share()

// 保存罚单
ZPDTicket.save()
```
