## 🚀 使用指南

### 基础使用

**自动触发** (无需配置):
1. 访问网站
2. 点击文章标题 → 朱迪彩蛋
3. 点击作者信息 → 尼克彩蛋
4. 点击页脚 → 闪电彩蛋

**手动触发**:
```javascript
// 在浏览器控制台
ztTriggerEgg('judy')   // 朱迪彩蛋
ztTriggerEgg('nick')   // 尼克彩蛋
ztTriggerEgg('flash')  // 闪电彩蛋
```

**键盘快捷键**:
```
Ctrl + Shift + E  →  随机彩蛋
```

### 高级功能

**查看所有角色**:
```javascript
ZtEasterEggs.getCharacters()
// ['judy', 'nick', 'flash', 'bogo', 'mayor']
```

**检查状态**:
```javascript
ZtEasterEggs.isEnabled()
// true 或 false
```

**禁用彩蛋**:
```javascript
ztDisableEggs()
// 或
localStorage.setItem('zt_eggs_disabled', 'true')
```

**启用彩蛋**:
```javascript
ztEnableEggs()
// 或
localStorage.removeItem('zt_eggs_disabled')
```
