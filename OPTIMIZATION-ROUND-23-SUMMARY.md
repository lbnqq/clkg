- 🎈 儿童节 (06-01)
- 🎃 万圣节 (10-31)
- 🎄 圣诞节 (12-25)

**节日横幅效果**:
- 渐入动画
- 自动消失（10秒）
- 手动关闭
- 重大节日徽章

#### 3. 轻量级实现 ⭐⭐⭐

- 仅12KB代码
- 粒子数量限制（最多15个）

- 使用 `requestAnimationFrame` 优化

## 🎨 功能特点

### 自动模式

```javascript
// 无需配置，自动检测
// 访问网站时自动生效 ✨
```

**工作流程**:
1. 检测当前月份 → 确定季节
2. 检测当前日期 → 确定是否有节日
3. 应用节日主题（优先）
4. 应用季节主题
5. 显示装饰效果
6. 显示季节消息

### 手动模式

```javascript
// 用户可手动切换季节
ztSetSeason('spring')  // 春季 🌸
ztSetSeason('summer')  // 夏季 ☀️
ztSetSeason('autumn')  // 秋季 🍂
ztSetSeason('winter')  // 冬季 ❄️
```

### 开发者模式

```javascript
// 显示自定义节日横幅
ZtSeasonal.showEventBanner({
    name: '我的生日',
    icon: '🎂',
    theme: 'spring',
    isMajor: true
});
```

## 📝 API文档

### 基础API

```javascript
// 获取当前季节
const season = ztGetSeason();  // 'spring' | 'summer' | 'autumn' | 'winter'

// 切换季节
ztSetSeason('winter');         // 切换到冬季
```

### 高级API

```javascript
// 获取所有季节配置
const seasons = ZtSeasonal.getSeasons();

// 获取当前节日
const event = ZtSeasonal.getCurrentEvent();

// 显示自定义横幅
ZtSeasonal.showEventBanner({
    name: '庆祝日',
    icon: '🎉',
    theme: 'spring',
    isMajor: false
});

// 刷新系统
ZtSeasonal.refresh();
```
