## 📊 技术实现

### 文件结构

```
第25轮新增文件 (2个):
├── zootopia-zpd-ticket.js     (~8KB) - ZPD罚单系统
└── zootopia-zpd-ticket.css    (~6KB) - 罚单样式
```

### 违规配置

```javascript
VIOLATION_TYPES = {
    traffic: {          // 交通违规
        fines: [5种违规]
    },
    public_disturbance: { // 扰乱公共秩序
        fines: [5种违规]
    },
    tax_evasion: {       // 税务违规
        fines: [3种违规]
    },
    fun: {               // 趣味违规
        fines: [5种违规]
    }
}
```

### API接口

```javascript
// 显示罚单生成器
ztShowZPDGenerator()

// 快速生成罚单
ztGenerateZPDTicket({
    violator: '尼克',
    type: 'traffic',
    location: '撒哈拉广场'
})

// 高级API
ZPDTicket.showGenerator()   // 显示生成器
ZPDTicket.generate()         // 生成罚单
ZPDTicket.print()           // 打印罚单
ZPDTicket.share()           // 分享罚单
ZPDTicket.save()            // 保存罚单
ZPDTicket.quickGenerate()   // 快速生成
```

## 🎨 使用场景

### 1. 互动罚单生成

**场景**: 访客可以生成有趣的ZPD罚单

**操作**:
1. 点击右下角👮浮动按钮
2. 填写违规信息（或留空随机）
3. 点击"生成罚单"
4. 查看罚单预览
5. 选择打印/分享/保存

### 2. 快速罚单

```javascript
// 在浏览器控制台快速生成
ztGenerateZPDTicket({
    violator: '朱迪',
    type: 'fun',
    location: 'ZPD总部'
})
```

### 3. API调用

```javascript
// 高级用法
const ticket = ZPDTicket.quickGenerate({
    violator: '尼克',
    type: 'traffic',
    location: '撒哈拉广场'
});
console.log(ticket);
```
