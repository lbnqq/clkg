## 🚀 使用指南

### 基础使用

**自动触发**: 无需任何操作，系统会自动在适当时机显示对话

**手动关闭**: 点击气泡右上角的×按钮

**查看更多**: 滚动、阅读、停留都会触发新对话

### 高级用法

```javascript
// 显示指定对话
ztShowDialogue('judy', 'Try Everything!')

// 显示随机对话
ztShowRandomDialogue('nick')

// 显示对话并自定义位置
ztShowDialogue('flash', '你...好...啊...', {
    position: 'top-right',
    autoClose: false
})

// 清除所有对话
ztClearDialogues()
```

### API调用

```javascript
// 高级API
ZTDialogue.show('judy', 0)           // 显示对话
ZTDialogue.showRandom('nick')        // 随机对话
ZTDialogue.clear()                   // 清除所有
ZTDialogue.getCharacters()           // 获取角色
ZTDialogue.getDialogues('judy')      // 获取台词
ZTDialogue.getConfig()              // 获取配置
```
