# 欢迎来到我的个人博客!

这是我使用 **Hexo** 博客框架创建的第一篇文章。在这篇文章中,我将分享一些关于博客搭建的经验。

## 为什么选择 Hexo?

Hexo 是一个快速、简洁且高效的博客框架,具有以下优点:

- **快速生成**: 基于Node.js,生成静态网站速度极快
- **丰富主题**: 拥有大量精美的主题可供选择
- **简单部署**: 可以轻松部署到 GitHub Pages 等平台
- **Markdown 支持**: 原生支持 Markdown 语法
