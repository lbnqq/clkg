          Hexo 博客便捷管理工具 使用说明
═══════════════════════════════════════════════

📁 工具文件说明:

1. 【新建文章.bat】 - 快速创建新文章
   ├─ 双击运行
   ├─ 输入文章标题
   └─ 自动打开编辑器

2. 【发布到GitHub.bat】 - 一键发布博客
   ├─ 编辑完文章后运行
   ├─ 自动清理、生成、部署
   └─ 完成!

3. 【本地预览.bat】 - 在本地查看博客
   ├─ 双击运行
   ├─ 浏览器访问: http://localhost:4000/cykg.github.io/
   └─ 按 Ctrl+C 停止

4. 【删除文章.bat】 - 删除不需要的文章
   ├─ 双击运行
   ├─ 显示所有文章列表
   └─ 选择要删除的文章

═══════════════════════════════════════════════

📝 典型使用流程:

【写作新文章】
Step 1: 双击 "新建文章.bat"
Step 2: 输入文章标题
Step 3: 在打开的编辑器中写内容
Step 4: 保存文件

【发布到线上】
Step 5: 双击 "发布到GitHub.bat"
Step 6: 等待完成
Step 7: 刷新 https://lbnqq.github.io/cykg.github.io/

【本地预览】(可选)
- 双击 "本地预览.bat"
- 在浏览器中预览效果
- 按 Ctrl+C 停止预览

【删除文章】
- 双击 "删除文章.bat"
- 按照提示删除

═══════════════════════════════════════════════

💡 推荐的 Markdown 编辑器:

1. Typora (收费但最好用)
2. VS Code + Markdown 插件 (免费)
3. Obsidian (免费)
4. Mark Text (免费开源)

═══════════════════════════════════════════════

⚠️ 注意事项:

1. 文章文件名建议使用中文或英文
2. 编辑时保持 Markdown 语法格式
3. 发布前记得保存文件
4. 每次修改后都需要运行"发布到GitHub.bat"
